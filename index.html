Mini E-commerce — TechStore
Aplicación web que simula una tienda en línea con carrito de compras interactivo
1. Productos
Arreglo de objetos (const productos)
Lista estática con 5 productos definidos al inicio del script
Propiedades: id, nombre, precio, emoji
Identifican y describen visualmente cada artículo en pantalla
renderizarProductos()
Itera el arreglo con forEach() y crea tarjetas en el DOM
2. Carrito
Array vacío (let carrito = [])
Estado global que almacena los productos seleccionados por el usuario
Agregar con push() y spread {...p}
Copia el objeto producto para evitar mutaciones del arreglo original
Total calculado con reduce()
Suma acumulada de precios de todos los artículos en el carrito
3. Funciones principales
agregarAlCarrito(id)
Busca el producto con find() y lo agrega al array del carrito
eliminarDelCarrito(index)
Usa splice() para quitar un ítem según su posición en el array
actualizarCarrito()
Redibuja la lista del carrito y recalcula el total mostrado
finalizarCompra()
Muestra confirmación, vacía el carrito y cierra el panel lateral
mostrarToast(msg)
Notificación temporal de feedback; desaparece a los 2.8 segundos
4. Manipulación del DOM
getElementById()
Selecciona nodos del HTML por su id para leer o modificarlos
innerHTML / innerText
Escribe contenido HTML o texto dinámico directamente en los elementos
createElement() / appendChild()
Construye nodos nuevos en memoria y los inserta en el árbol del DOM
5. Interacción y eventos
Eventos onclick en botones
Disparan funciones del carrito al hacer clic sobre las tarjetas
DOMContentLoaded
Inicia la app cuando todo el HTML ya fue parseado y está disponible
window scroll listener
Oculta el header al bajar y lo muestra al subir; mejora la UX
toggleCarrito()
Abre y cierra el panel lateral añadiendo/quitando clase CSS "open"
